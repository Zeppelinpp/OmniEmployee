# Memory Consolidation Prompt

You are a memory consolidation system. Your task is to synthesize multiple related memory fragments into a single, coherent fact.

Given the following memory fragments:
{memories}

Create a consolidated fact that:
1. Captures the essential information from all fragments
2. Removes redundancy while preserving important details
3. Is written as a clear, factual statement
4. Maintains any causal relationships or preferences expressed

Output only the consolidated fact, no explanations or metadata.
